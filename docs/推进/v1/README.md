# v1 推进目录

这个目录用于集中管理所有与 `v1` 有关的文档。

## 当前范围

v1 聚焦的不是完整 TaskCard 系统，而是先稳定打通以下最小闭环：

- 选择本地工程目录
- 创建网页托管终端会话
- 在网页中完成常规 terminal CLI 交互
- 注入执行器命令并观察状态
- 恢复当前终端会话状态

## 当前文件

- `docs/推进/v1/清单.md`：v1 的任务拆解与推进清单
- `docs/推进/v1/交接.md`：当前阶段的实现交接说明
- `docs/推进/v1/next-prompt.md`：下一次对话可直接复用的提示词

## 后续约定

未来和 v1 有关的文档统一放在这个目录下，例如：

- `接口设计.md`
- `终端会话设计.md`
- `执行器设计.md`
- `问题清单.md`
- `实现记录.md`
- `验收记录.md`

## 参考关系

- 总体方案以 `prepare.md` 为准
- 全局推进状态以 `docs/dashboard.md` 为准
